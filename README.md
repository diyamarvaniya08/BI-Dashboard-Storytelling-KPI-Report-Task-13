# BI-Dashboard-Storytelling-KPI-Report-Task-13
Interactive Power BI dashboard analyzing coffee shop sales data to track revenue trends, transaction patterns, payment methods, and top-performing coffee products using data cleaning, transformation, and visual storytelling techniques.

# Task Overview
- This task focuses on analyzing coffee shop sales data and building an interactive dashboard using Power BI. The objective is to understand sales performance, customer behavior, and revenue trends through meaningful visualizations and business insights.

# Dataset Used
 - Dataset Name: Coffee Shop Sales
 - File Format: Excel (.xlsx)

    ## Key Columns:
   - Transaction Date
   - Transaction Quantity
   - Unit Price
   - Coffee Type / Product
   - Sales (calculated column)

# Tools & Technologies Used
 - Power BI
 - Power Query Editor

# Work Performed
 - Cleaned and transformed raw data using Power Query
 - Created a new Sales column using transaction quantity
 - Handled missing values and corrected data types
 - Structured the dataset for effective reporting
 - Applied a coffee-themed color palette for better visual appeal

# Dashboard Features
 - Total Sales, Orders, and Quantity KPI cards
 - Monthly Sales Trend (line chart)
 - Sales by Product Category (bar chart)
 - Top-Selling Coffee Products
 - Day-wise Sales Performance
 - Interactive filters and slicers for deeper analysis

# Key Insights
 - Identified peak sales months and high-performing days
 - Determined the most popular coffee categories
 - Analyzed customer payment preferences
 - Highlighted top revenue-generating products

# Learning Outcomes
 - Hands-on experience with real-world sales data
 - Understanding data cleaning and transformation workflow
 - Building interactive dashboards with business focus
 - Improving data storytelling and insight generation skills
